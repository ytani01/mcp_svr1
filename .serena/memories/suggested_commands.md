# Suggested Commands

## Installation

```bash
git clone https://github.com/ytani01/mcp_svr1
```

## Running the MCP Server

```bash
uv run mcp_svr1
```

## Linting

```bash
uv run ruff check .
uv run mypy .
uv run pyright .
```

## Testing

```bash
uv run python -m pytest -v
```

## Adding a new library

```bash
uv add <library-name>
```
